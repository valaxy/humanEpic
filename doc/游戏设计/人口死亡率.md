# 使用逻辑斯谛函数模拟人口死亡率

使用逻辑斯谛函数（Logistic Function）来模拟死亡率是一个非常巧妙的思路。通常我们用它来描述生长（比如人口增长的 S 型曲线），但如果要模拟**死亡率**，我们需要将其反向运用：当食物满足度降低时，死亡率升高；当食物满足度充足时，死亡率降低并趋于稳定。

### 1. 模型逻辑构建

我们要建立的是**食物满足度 ($s$)** 与 **死亡率 ($d$)** 之间的关系。

* **自变量 ($s$)**：食物满足度，范围在 $[0, 1]$。
* **因变量 ($d$)**：死亡率。
* **逻辑趋势**：
    * 当 $s \to 0$（极度饥饿）时，死亡率应该迅速飙升至最大值。
    * 当 $s \to 1$（食物充足）时，死亡率应该下降并维持在一个自然基准水平（老死、疾病等）。

---

### 2. 数学公式定义

为了实现这种“负相关”的 S 型曲线，我们可以使用逻辑斯谛函数的反向变体：

$$d(s) = d_{min} + \frac{d_{max} - d_{min}}{1 + e^{k(s - s_0)}}$$



---

### 3. 参数的物理意义

为了让模拟更真实，可以通过调整以下四个参数来控制曲线形态：

| 参数 | 名称 | 物理含义 |
| :--- | :--- | :--- |
| **$d_{max}$** | **最大死亡率** | 当食物几乎为零时的理论最高死亡率（例如 1.0）。 |
| **$d_{min}$** | **基准死亡率** | 食物完全充足时的自然死亡率（例如 0.01）。 |
| **$k$** | **陡度系数** | 死亡率对食物变化的**敏感程度**。$k$ 越大，曲线越陡。 |
| **$s_0$** | **拐点/中值** | 死亡率变化最剧烈的食物临界点。通常设在 0.3 - 0.5 之间。 |

---

### 4. 模拟示例

假设我们设定以下数值：
* $d_{max} = 0.8$（极度饥饿时，80% 的人会死亡）
* $d_{min} = 0.02$（食物充足时，只有 2% 的自然死亡）
* $s_0 = 0.3$（当食物满足度低于 0.3 时，死亡率开始剧增）
* $k = 15$（敏感度较高）

**具体的计算公式为：**
$$d(s) = 0.02 + \frac{0.78}{1 + e^{15(s - 0.3)}}$$

---

### 5. 模型的逻辑优势

1.  **有界性**：死亡率永远不会低于 $d_{min}$，也不会超过 $d_{max}$，符合生物学常识。
2.  **耐受极限**：它模拟了生物的“缓冲期”。在食物从 1.0 降到 0.7 时，死亡率变化很小；但一旦跌破 $s_0$，死亡率会发生崩塌式上升。
